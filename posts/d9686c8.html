3.HTML样式-CSS

CSS：级联样式表–用于格式化网页布局；层叠–继承

使用方式：

​ 内联：使用styleHTML元素内在属性；将唯一样式应用于单个属性

​ 内部：使用 <head>元素存储 <style>元素控制的其余元素的格式；应用于单个HTML页面

​ 外部：使用 <head>元素存储<link>元素控制指定的外部CSS文件的链接

style.css

CSS color属性定义文本颜色

CSS font-fanmily属性定义文本字体

CSS font-size属性定义文字大小

CSS font-align属性定义水平文本对齐方式

CSS border属性定义HTML元素边框；尺寸+颜色

CSS padding属性在边框内流出空间

CSS margin属性定义边框外的边距（空格）

链接到外部CSS

4.HTML链接

HTML超链接：<a>元素的 <href>元素指示链接的地址

<target>属性指定在何处打开链接的文档；更改默认显示在当前浏览窗口的设置

  • _self-默认。在与单击相同的窗口/选项卡中打开文档
  • _blank-在新窗口/标签中打开文档
  • _prent-在父框架中打开文档
  • _top-在整个窗口打开文档

<a href="https://www.w3schools.com/" target="_blank">Visit W3Schools!</a>

绝对URL：href中使用完整的Web地址

相对URKL：不带"https://www部分"

使用图像作为链接：将 <img>标签放在标签 <a>内

链接到邮件地址：mailot在 href属性内使用可创建一个打开邮箱的链接

<a href="mailto:someone@example.com">Send email</a>

按钮作为链接：使用JS

<button onclick="document.location='default.asp'">HTML Tutorial</button>

链接标题：title属性指定有关元素额外（提示）信息

链接默认状态下的颜色显示：

  • 未访问的链接带有下划线且为蓝色
  • 已访问的链接带有下划线且为紫色
  • 活动的链接带有下划线且为红色

创建书签：

​ id属性创建书签；id =“ value ”

​ <h2 id="C4">Chapter 4</h2>

​ 然后从同一页面添加到书签的链接（跳到第4章）

​ <a href="#C4">Jump to Chapter 4</a>

​ 从另一个页面添加指向书签的链接；href =“＃value”

​ <a href="html_demo.html#C4">Jump to Chapter 4</a>

5.HTML图片

HTML图像标签：<img>,<map>,<area>,<picture>

<1>.普通图片

<img>标签创建被引用的图像的容纳空间；只包含属性

  • src-指定图像的路径；需要确保图像相对于网页处于同一位置
  • alt-图像无法加载时指定的替代文字

<img src="url" alt="alternatetext">

图像尺寸

  • style属性指定图像尺寸

<img src="img_girl.jpg" alt="Girl in a jacket" style="width:500px;height:600px;">

  • width和 height属性

<img src="img_girl.jpg" alt="Girl in a jacket" width="500" height="600">

另一个文件夹中的图像：src属性包括文件夹名称

另一台服务器/网站上的图像：src属性指定绝对URL

用图像做链接：将 <img>标记放在标记 <a>内

图像浮动：使用CSS的 float属性使图像浮动到文本一侧

<2>.影像图

<map>标记定义图像映射。图像地点是具有可点区域的图像；用 <area>标签定义区域

影像图工作原理

  • 使用 <img>标签插入图像，唯一区别为必须添加 usemap属性

    <img src="workplace.jpg" alt="Workplace" usemap="#workmap">

    #号用于创建图像和图像地图之间的关系

  • 添加一个 <map>元素，用于创建图像映射且通过所需要的链接到图像属性

    <map name="workmap"> name属性值对应于usemap属性

  • 通过 <area>添加可点击区域

  • 定义形状（需要指定坐标点）

    • rect-定义一个矩形区域；shape="rect"成对存在对应x和y轴
    • circle-定义一个圆形区域；shape="circle"指定圆心坐标和半径
    • poly-定义一个多边形区域；Shape =“ poly”包含多边形所有边缘坐标点
    • default-定义整个区域

    <area shape="rect" coords="34, 44, 270, 350" href="computer.htm">

    <area shape="circle" coords="337, 300, 44" href="coffee.htm">

影像图可点击区域可以出发JS函数：click向 <area>元素添加事件执行JS函数

<3>HTML背景图

  • 使用 style属性和CSS background-image属性

<div style="background-image: url('img_girl.jpg');">

  • 在<style>元素的 <head>部分指定背景图片

  • 为整个页面添加背景图片；在 <body>元素上指定背景图像

  • 为避免背景图像重复出现，请将background-repeat属性设置为no-repeat

背景封面：

​ background-size: cover

​ 如果希望背景图像覆盖整个元素，则可以将background-size属性设置为 cover.

​ 另外，要确保始终覆盖整个元素，请将background-attachment属性设置 为fixed:(图像维持原始比例)

<4>图片元素

<source>元素含有 media属性定义何时最适合图像；<img>元素作为备用元素

浏览器或设备使用第一种识别的图像格式
